Cursor RTL

תמיכת RTL רב-לשונית חכמה ל-Cursor AI Chat, חלון Agents וקבצי Plan. אלגוריתם ייעודי מזהה את כיוון הטקסט לכל אלמנט ומשאיר תוכן אנגלי מיושר לשמאל.

⚡

התקנה בלחיצה אחת

הפעלה או כיבוי תמיכת RTL מלוח הפקודות בפקודה אחת. ללא עריכה ידנית של קבצים.

🔤

זיהוי אוטומטי חכם

כל אלמנט מנותח בנפרד עם ניקוד RTL/LTR משוקלל, כך שטקסט עברי, ערבי ופרסי מיושר לימין ותוכן אנגלי נשאר לשמאל.

🔄

שורד עדכונים

מזהה כשעדכוני Cursor דורסים את התיקון ומציע להחיל מחדש אוטומטית. ההגדרות שלך נשארות.

📝

תמיכה בקבצי Plan

תומך בעורך ה-rich text של קבצי .plan.md, כולל כללי CSS ממוקדים לתוכן ש-Cursor מייצר מחדש.

🤖

חלון Agents

מרחיב את RTL לחלון Agents, כולל שיחת הסוכן ותצוגת ה-Plan הצדדית.

▦

טבלאות Markdown

מטפל בטבלאות Markdown מעורבות לפי כיוון רוב התאים, כולל בתוך קבצי Plan.

⌘

מזהה קוד ומונחים טכניים

נתיבים, שמות API ומזהים באנגלית מקבלים משקל נמוך יותר, כדי שטקסט RTL עם קוד לא יתהפך בטעות.

🛡

החלה בטוחה

הטעינה מתבצעת רק בחלונות workbench, עם גיבוי, rollback ופעולות idempotent.

התחלה

התקנה ב-4 צעדים

1

הורדה

הורידו את ה-.vsix האחרון מדף ה-Releases.

2

התקנת התוסף

פתחו את תצוגת התוספים (Ctrl+Shift+X) → לחצו על תפריט ... → בחרו Install from VSIX... → בחרו את הקובץ שהורדתם.

3

הפעלת RTL

חפשו את RTL: OFF בצד ימין של שורת המצב התחתונה → לחצו → בחרו Enable RTL.

4

הפעלה מחדש

אשרו את ההתקנה → סגרו ופתחו מחדש את כל חלונות Cursor.
